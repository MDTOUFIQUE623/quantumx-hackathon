In plain English: Think of a glass wall at the polling station showing “Ballot #7 received, Ballot #8 received…” without showing who voted or how they voted. Everyone can see the list grew honestly; only scrambled content is visible.

Technical (demo): This page shows receipt numbers and scrambled ballot data — not voter names. The server still remembers who voted (to stop double voting); a full production system could add an extra shuffle so even the server cannot link person to ballot.
